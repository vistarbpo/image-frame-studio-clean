# Image Frame Studio

A modern web application for adding beautiful frames to your images.

## About

Developed by [Salim](mailto:salimthottupoyil@gmail.com) at Sherikah Company.

## Features

- Image upload and manipulation
- Custom frame selection
- Real-time preview
- Download framed images
- Responsive design

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Shadcn UI
- React Router
- React Query

## Getting Started

### Prerequisites

- Node.js 18 or higher
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/image-frame-studio.git
cd image-frame-studio
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```

## Deployment

This project is configured for deployment on Vercel:

1. Push to GitHub
2. Import to Vercel
3. Deploy

## Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_URL=your_api_url
```
